The data is required from the selected LFIs for complaints received through various channels, for the period defined by the Central LFI of UAE. This information is required solely from the respective LFI and should not encompass any data related to group companies or subsidiaries.
The Complaints Data Tab consists of 2 categories of inputs- 1. Drop down list - Select the most appropriate response 2. Free text fields - Enter the required input
Notes:- The LFI is required to update all blank fields provided for each data element. If a certain field is greyed out, the LFI is not required to update the respective data field.
Exceptions - 1. In scenarios where a certain data element is not applicable (Ex - The LFI does not offer the respective product) the LFI should update the Remarks field as Not Applicable. 2. In scenarios where the LFI is not able to provide the requested data for any reason (such as - LFI does not maintain data, loss of data due to system crashes) the LFI should update the Remarks field as Not Available. 3. In scenarios where a certain data element is applicable on the LFI however there is no occurrence of the respective requirement the LFI should update the respective field with the numeric value of zero ( 0 ).
